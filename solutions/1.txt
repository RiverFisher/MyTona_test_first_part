Вышеприведённый код содержит следующие ошибки.

1) Для выделения памяти под одномерный динамический массив
синтакисис языка требует использовать квадратные скобки
с верхней границей размерности внутри них:
// ... j = new int[20];

2) В цикле указан обход 21-го элемента массива вместо 20-ти.
Следует или заменить нестрогое условие строгим, или уменьшить на единицу
верхнюю границу итератора:
// for (int i = 0; i < 20; i++) {

// for (int i = 0; i <= 19; i++) {

Кроме того, ниже приведены рекомендации к рефакторингу кода.

1) Нет необходимости в явном указании типа переменной j. Используйте спецификатор auto:
// auto j = new int[20];

2) После того, как нет необходимости в динамическом массиве,
следует очистить память, выделенную под него:
// delete [] j;

Таким образом, исправленный код может быть таким:

int main(int argc, char** argv) {
    auto j = new int[20];
    for (int i = 0; i < 20; i++) {
        j[i] = i;
    }

    delete [] j;

    return 0;
}
