Мы все понимаем, что просьба указать очерёдность выполнения операций относится к строке
// bool result = a+++b ? 0 : 0 || 0 && 1;
Поэтому я указываю очерёдность выполнения операций только для этой строки.
Очерёдность выполнения операций:

1) a++ не сразу инкрементирует значение переменной a, поэтому сперва будет выполнен оператор
   a  +b - вернёт 0;
2) a++ - постинкрементирует переменную a, a станет равной 1;
3) неявное преобразование 0 к логическому типу данных bool - вернёт false;
4) a+++b ? /* 'true'-branching */ : /* 'false'-branching */ - результат выполнения условия в
   тернарном операторе станет равным 0, будет выполнена 'false'-branching ветка:
   > неявное преобразование 0 к логическому типу данных bool - вернёт false;
   > неявное преобразование 1 к логическому типу данных bool - вернёт true;
   > false && true - вернёт false;
   > неявное преобразование 0 к логическому типу данных bool - вернёт false;
   > false || false - вернёт false;
5) bool result - инициализация переменной result;
6) result = ... - присвоение переменной result результата выполнения действий пункта 3)
   result равна false;
